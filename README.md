# Korroziyani Kamaytirish Yo'llari

Ushbu sayt metallarning korroziyadan himoyalash usullari haqida to'liq ma'lumot beradi. Zamonaviy dizayn va interaktiv diagrammalar bilan korroziya jarayonlarini tushunish osonlashadi.

## Xususiyatlari

- **Himoya usullari**: Qoplama, katodik himoya, ingibitorlar
- **Interaktiv diagrammalar**: Korroziya progressiyasi, himoya usullari samaradorligi
- **Kimyoviy formulalar**: Reaktsiyalar va jarayonlarning matematik ifodasi
- **pH ta'siri**: Turli metallarning pH ga bog'liq korroziya tezligi

## Texnologiyalar

- HTML5 semantik tuzilma
- Tailwind CSS dizayn
- Chart.js interaktiv diagrammalar
- Font Awesome ikonlar
- Responsive dizayn

## Foydalanish

1. `index.html` faylini brauzerda oching
2. Bo'limlar orasida navigatsiya qiling
3. Diagrammalar bilan tanishining
4. Kimyoviy jarayonlarni o'rganing

## Tuzilma

```
├── index.html          # Asosiy sahifa
└── README.md          # Hujjatlar
```

Sayt to'liq funksional va barcha zamonaviy brauzerlarda ishlaydi.
